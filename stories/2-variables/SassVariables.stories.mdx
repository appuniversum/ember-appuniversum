# SCSS variables

## Global settings

```css
// Breakpoints
// ---
$mq-breakpoints: (
  tiny: 300px,
  xsmall: 500px,
  small: 767px,
  medium: 1023px,
  large: 1600px
) !default;
// Uncomment line below to show breakpoint indicators
// $mq-show-breakpoints: (xsmall, small, medium, large);

// Spacing units
// ---
$au-unit-tiny: 0.6rem !default;
$au-unit-small: 1.2rem !default;
$au-unit: 2.4rem !default;
$au-unit-large: 4.8rem !default;
$au-unit-huge: 9.6rem !default;

// Objects settings
// ---
// Layout
$au-layout-width: 1280px !default;
$au-layout-width-small: 768px !default;
$au-layout-width-large: 1600px !default;
$au-layout-breakpoint: medium !default;

// Region
$au-region-breakpoint: medium !default;
```
