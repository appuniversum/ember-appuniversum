## Icons

The appuniversum icon component is a wrapper around [Ember SVG Jar]('https://www.npmjs.com/package/ember-svg-jar').

### Usage as a symbolset (default)

Recommended when there's a lot of repetition.

Set the strategy to 'symbol'

```js
var app = new EmberApp(defaults, {
   svgJar: {
      strategy: 'inline',
   }
});
```

### Inlining svg's

Recommended when there's no repeating icons.

Set the strategy to 'symbol'

```js
var app = new EmberApp(defaults, {
   svgJar: {
      strategy: 'inline',
   }
});
```

### Additional options

Add extra locations for custom application icons. 

You can also mix the symbol and inline strategy.

For more information see the [ember-svg-jar documentation](https://github.com/ghedamat/ember-svg-jar/blob/master/packages/ember-svg-jar/docs/configuration.md).

```js
var app = new EmberApp(defaults, {
   strategy: ['symbol', 'inline'],

   symbol: {
      sourceDirs: ['public/images/svg/icons']
   },

   inline: {
      sourceDirs: ['public/images/svg/illustrations']
   }
});
```
