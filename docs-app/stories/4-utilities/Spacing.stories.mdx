Each spacing utility can be made responsive by adding a breakpoint extension.

`.au-u-margin-bottom-small@xsmall`

`.au-u-margin-bottom-small@small`

`.au-u-margin-bottom-small@medium`

`.au-u-margin-bottom-small@large`
